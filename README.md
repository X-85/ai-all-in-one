# AI 全栈知识库

> 记录 AI 学习过程，包括 Agent、Skills、Sub-Agent 等相关知识

## 📚 内容

- **基础知识** - 环境配置等基础内容
- **临时笔记** - 学习过程中的笔记和文档

## 🚀 部署

本知识库使用 [docsify](https://docsify.js.org/) 构建，托管在 GitHub Pages 上。

## 📝 编辑指南

1. 所有文档使用 Markdown 格式
2. 文档放在 `knowledge/` 目录下
3. 编辑完成后提交到 GitHub

```bash
git add .
git commit -m "更新文档"
git push
```
